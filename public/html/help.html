How To Play The Game

The area in red is known as the game window. In order to start the game, click on the game window. It is also possible to stop the game by click on the same area. The game can continued from where it was stopped by clicking on the same area. Doing this will not restart the game. However, if the game is lost or won, then the game will start over if the area is once again clicked.
In order to move the paddle left or right, simple move the mouse left or right on the page. If the paddle reaches the edge of the game window, it will not progress any further. You will need to move the mouse the opposite direction until the paddle is able to move once more. Note that moving the mouse up or down does not have any effect on the game. Additionally, where the ball strikes the paddle will an effect on how fast the ball moves left or right. If the ball strikes the paddle where the red mark is, it will not move left or right. The further left or right of the paddle where the ball strikes, determines how fast the ball will move in that direction.
The current game level will be displayed in the top left hand corner of the game area. When a player destroys all of the bricks the game will automatically proceed to the next level. Each level contains a different configuration of bricks. At the start of each new level, the level number will be displayed in the center of the game window. The player can click anywhere within the game window to start the new level. The ball will move faster with each new level in order to increase the difficulty of the game.
The current score will be displayed in the center of the screen above the game window. The player will earn 1 point each time the game ball hits a brick. The red bricks require 3 hits to destroy. The yellow bricks require 2 hits to destroy. And the blue bricks require 1 hit to destroy.
Each player will start with 3 lives. The current number of lives will be listed in the top right hand corner of the game area. If the ball strikes the bottom boundary of the game window, the player will lose 1 life. Additionally, the current number of lives will be displayed in the game window. The player will click anywhere within the game window in order to resume playing the game. The game will continue until a player has no lives left. If the player's score qualifies as a high score a window will prompt the user to enter their name. The message "Game Over" will be displayed in the game window.
If the game is becoming too difficult, then the cheat mode can be activated. This is done by click on the button labeled "Enable Cheats". Once this is done, the button text will change to "Disable Cheats". The cheats can be disabled or enabled at any time. When cheats are enabled, the ball will pierce multiple bricks, before bouncing off the edge of the game window. Additionally, when cheats are enabled, the ball will bounce of the bottom of the screen, instead of ending the game.
